# Mic-Injetor-Linux-
Script de automação para roteamento de áudio de baixo nível. Captura o sinal bruto do microfone diretamente via ALSA (Kernel) e o injeta em uma linha virtual dedicada no Pipewire/PulseAudio. Ideal para hardware USB (como o ME6S) que apresenta instabilidade ou não é detectado pelas interfaces gráficas padrão.
